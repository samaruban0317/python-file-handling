# Python File Handling

This repository contains Python scripts written while learning
file handling, lists, dictionaries, and tuples using the
Python for Everybody specialization (University of Michigan – Coursera).

## Files
- email_hour_distribution.py – Email hour-wise analysis
- list_senders.py – Extracts sender email IDs
- most_frequent_sender.py – Finds the most frequent sender
- email_logs_sample.txt – Sample email log file

## Purpose
Learning-focused practice code.
